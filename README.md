# Demiurge

**Demiurge** is a meta-agent — a *Thrallherd* — that mints, specs, delegates to, and curates
specialized **Archon** agents. It owns exactly one loop: **agent creation and curation**.

It is deliberately **not**:

- a runtime,
- an agent format,
- a protocol.

Instead it rides the existing rails — agent format standards, **A2A** for agent-to-agent
communication, and **MCP** for tool access — and focuses solely on the layer above them:
deciding *what agent should exist*, specifying it, delegating work to it, and curating the
resulting stable of Archons over time.
