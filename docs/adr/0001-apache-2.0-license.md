# ADR 0001 — License: Apache-2.0

- **Status:** Accepted (2026-07-06)

## Context

Demiurge is a public, portfolio-grade open-source project (see `PLAN.md`). The license should
invite adoption by other builders while protecting contributors.

## Decision

Apache License 2.0.

## Consequences

- Permissive enough for anyone to build on Demiurge, commercially or otherwise.
- The explicit patent grant makes it the conventional choice for infrastructure projects and
  signals maintenance seriousness better than MIT.
- Derivative works must carry attribution and note changes; no copyleft obligations.
